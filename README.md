# 🚀 ResumeIQ

> AI-Powered Resume Analyzer built with FastAPI + React.

ResumeIQ is a modern web application that analyzes resumes, calculates ATS scores, compares resumes with job descriptions, and provides actionable recommendations to improve hiring success.

---

## 🌐 Live Demo

**Frontend:** https://resumeiq-mu.vercel.app

**Backend API:** https://resumeiq-5e2z.onrender.com

---

## ✨ Features

- 📄 Resume upload (PDF & DOCX)
- 🤖 AI-powered resume analysis
- 🎯 ATS score calculation
- 💼 Job description matching
- ✅ Skill detection
- ⚠️ Missing skills identification
- 💡 Personalized recommendations
- 📊 Dashboard analytics
- 📚 Analysis history
- 🌙 Dark / Light mode
- 📱 Responsive design

## ⚙️ Installation

### Backend

```bash
cd backend

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
